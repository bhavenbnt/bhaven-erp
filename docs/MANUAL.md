## 1. 시스템 개요

### 1.1 역할 구분

| 역할 | 설명 | 주요 업무 |
|------|------|----------|
| **고객사** (Customer) | 콜드브루 추출을 의뢰하는 업체 | 예약 신청, 진행 현황 확인, 문의 |
| **관리자** (Admin) | 비해이븐 관리 담당자 | 예약 승인/반려, 일정 관리, 고객/직원 관리 |
| **슈퍼 관리자** | 최상위 권한 관리자 | 관리자 기능 + 고객사/직원 계정 생성 |
| **작업자** (Worker) | 현장 생산 담당자 | 생산 작업 확인, 출고 처리 |

### 1.2 예약 상태 흐름

```
승인 대기 → 확정 → 생산 중 → 완료 → 출고 완료
   ↓
  취소/반려
```

| 상태 | 설명 |
|------|------|
| **승인 대기** | 고객이 예약 신청 후, 관리자 승인 대기 |
| **확정** | 관리자가 승인 완료, 생산일 대기 |
| **생산 중** | 생산일 당일, 자동 전환 (매일 정오) |
| **완료** | 생산 완료, 출고 대기 |
| **출고 완료** | 택배 발송 완료, 송장번호 입력됨 |
| **취소** | 고객 취소 또는 관리자 반려 |

### 1.3 장비 종류

| 종류 | 용량 | 분할 |
|------|------|------|
| 소형 | 1~10kg | 4분할 (슬롯당 최대 3kg) |
| 중형 | 10~20kg | 단일 사용 |
| 대형 | 25~60kg | 단일 사용 |
| 기타 | 직접 설정 | 단일 사용 |

## 2. 공통 기능

### 2.1 로그인

1. 이메일과 비밀번호를 입력합니다.
2. "로그인" 버튼을 클릭합니다.
3. 역할에 따라 자동으로 해당 대시보드로 이동합니다.
   - 고객사 → 고객 대시보드
   - 관리자 → 관리자 대시보드
   - 작업자 → 작업자 대시보드

### 2.2 회원가입 (고객사)

1. 로그인 화면 하단의 "회원가입" 링크를 클릭합니다.
2. 이메일, 비밀번호, 담당자명, 업체명, 연락처를 입력합니다.
3. 이용약관 및 개인정보 처리방침에 동의합니다.
4. "회원가입" 버튼을 클릭합니다.
5. **관리자 승인 후** 로그인이 가능합니다.

### 2.3 알림

- 화면 우측 상단의 🔔 아이콘을 클릭하면 알림 목록이 표시됩니다.
- 새 알림이 있으면 빨간 뱃지가 나타납니다.
- 알림을 클릭하면 해당 페이지로 이동합니다.
- "모두 읽음" 버튼으로 일괄 처리할 수 있습니다.

### 2.4 프로필 설정

1. 사이드바 하단의 "프로필 설정"을 클릭합니다.
2. 업체 정보(업체명, 담당자명, 연락처)를 수정할 수 있습니다.
3. 비밀번호 변경: 현재 비밀번호 → 새 비밀번호 → 확인 입력 후 "변경" 버튼을 클릭합니다.

## 3. 고객사 매뉴얼

### 3.1 대시보드

로그인 후 가장 먼저 보이는 화면입니다.

**표시 정보:**
- 진행 중인 예약 건수
- 승인 대기 건수
- 이번 달 완료 건수
- 최근 예약 목록

### 3.2 예약 캘린더

생산 가능한 날짜를 확인하고 예약을 신청할 수 있습니다.

**사용 방법:**
1. 사이드바에서 "예약 캘린더"를 클릭합니다.
2. 월별 캘린더에서 각 날짜의 가용 현황을 확인합니다.
   - 🟢 초록: 여유 있음
   - 🟠 주황: 보통
   - 🔴 빨강: 혼잡
   - 회색: 예약 불가 (휴무일 또는 가용 기기 없음)
3. **용량 필터**: 상단의 용량 프리셋(10kg, 20kg 등)을 선택하거나 직접 입력하면, 해당 중량을 처리할 수 있는 날짜만 표시됩니다.
4. 원하는 날짜를 클릭하면 예약 신청 페이지로 이동합니다.

**내 예약 확인:**
- 캘린더에서 내가 신청한 예약이 있는 날짜에는 "내 예약 N건" 뱃지가 표시됩니다.

### 3.3 예약 신청

새로운 생산 예약을 신청합니다.

**입력 항목:**

| 항목 | 필수 | 설명 |
|------|------|------|
| 품목 | 선택 | 원액 또는 캔 |
| 용기 | 선택 | 원액: 500ml/1L/2L, 캔: 235ml/330ml |
| 제품명 | ✅ | 예) 프리미엄 콜드브루 |
| 수율 | 선택 | L/kg 단위 (이전 수율 자동 표시) |
| 원두 중량 | ✅ | kg 단위 |
| 생산 희망일 | ✅ | 캘린더에서 선택 |
| 비고 | 선택 | 특이사항 기재 |

**편의 기능:**
- **이전 주문 불러오기**: 상단에 최근 주문한 제품이 표시됩니다. 클릭하면 제품명과 수율이 자동 입력됩니다.
- **자동 배정 미리보기**: 중량을 입력하면 어떤 기기에 배정될지 미리 확인할 수 있습니다.
- **예상 생산량**: 중량 × 수율로 계산된 예상 생산량이 표시됩니다.

**신청 후:**
- 예약이 "승인 대기" 상태로 등록됩니다.
- 관리자가 승인하면 "확정" 상태로 변경되며 알림이 발송됩니다.

### 3.4 내 예약 현황

신청한 모든 예약을 확인합니다.

**기능:**
- **상태별 탭**: 전체, 승인 대기, 확정, 생산 중, 완료, 취소
- **상세 보기**: 목록에서 예약을 클릭하면 상세 정보를 확인할 수 있습니다.
- **예약 취소**: 승인 대기 또는 확정 상태의 예약만 취소할 수 있습니다.

### 3.5 관리자 문의

관리자에게 문의사항을 전달합니다.

**새 문의 신청:**
1. "새 문의" 버튼을 클릭합니다.
2. 문의 유형, 제목, 내용을 입력합니다.
3. "문의 접수" 버튼을 클릭합니다.

**문의 확인:**
- 문의 목록에서 상태(접수/답변 완료)를 확인합니다.
- 클릭하면 관리자의 답변을 확인할 수 있습니다.

## 4. 관리자 매뉴얼

### 4.1 대시보드

**통계 카드:**
- 오늘 생산: 오늘 예정된 생산 건수
- 승인 대기: 고객이 신청하여 승인이 필요한 건수
- 이번 달 완료: 이번 달 생산 완료 건수
- 가동 기기: 현재 정상 가동 중인 기기 수

**승인 대기 목록:**
- 고객이 신청한 예약이 표시됩니다.
- **승인**: 버튼 클릭 시 즉시 "확정" 상태로 변경됩니다.
- **반려**: 버튼 클릭 시 반려 사유를 입력하는 모달이 표시됩니다. 사유 입력 후 확인하면 고객에게 알림이 발송됩니다.

### 4.2 예약 관리

모든 예약을 조회하고 관리합니다.

**검색 및 필터:**
- 상단 검색창: 업체명, 제품명, 예약번호로 검색
- 상태 탭: 전체/승인 대기/확정/생산 중/완료/취소
- 날짜 필터: 시작일~종료일 범위 지정

**예약 추가 (관리자 수동 생성):**
1. 우측 상단의 "예약 추가" 버튼을 클릭합니다.
2. 고객사 선택:
   - **기존 고객**: 검색 필터로 고객을 찾아 선택
   - **직접 입력**: 미등록 고객의 업체명을 직접 입력
3. 제품명, 원두 중량, 품목, 용기, 생산 희망일을 입력합니다.
4. **용량 검증**: 날짜와 중량을 입력하면 자동으로 해당일 가용 여부가 표시됩니다.
   - ✅ "해당일 생산 가능" → 바로 생성 가능
   - ⚠️ "해당일 용량 부족" → "용량 초과 강제 생성" 체크 후 생성 가능
5. "예약 생성" 버튼을 클릭합니다.
6. 관리자가 생성한 예약은 승인 대기 없이 바로 **확정** 상태로 등록됩니다.

**예약 상세:**
- 목록에서 예약을 클릭하면 상세 페이지로 이동합니다.
- 예약 승인/반려, 기기 재배정, 일정 변경이 가능합니다.

### 4.3 생산 캘린더

월별 생산 일정과 휴무일을 관리합니다.

**캘린더 보기:**
- 각 날짜에 예약된 생산 건이 기기명과 함께 표시됩니다.
- 기기 종류별 필터(전체/소형/중형/대형/기타)로 보기를 조절할 수 있습니다.
- 클릭하면 해당 날짜의 상세 생산 목록을 확인합니다.

**휴무일 관리:**
1. "휴무일 관리" 버튼을 클릭합니다.
2. 날짜를 선택하고 사유를 입력합니다.
3. 해당 날짜에 기존 예약이 있으면 경고 모달이 표시됩니다.
   - "N건의 예약이 있습니다. 휴무일로 등록하면 기존 예약은 관리자가 별도로 일정을 변경해야 합니다."
4. 확인을 누르면 휴무일이 등록됩니다.
5. 등록된 휴무일은 캘린더에 표시되며, 고객은 해당 날짜에 예약할 수 없습니다.
6. 휴무일 삭제: 해당 날짜의 ✕ 버튼을 클릭합니다.

### 4.4 기기 관리

생산 장비를 등록하고 관리합니다.

**기기 목록:**
- 모든 기기의 이름, 코드, 종류, 용량, 상태가 표시됩니다.

**기기 추가:**
1. "기기 추가" 버튼을 클릭합니다.
2. 기기명, 기기 코드, 종류(소형/중형/대형/기타), 최소/최대 용량을 입력합니다.
3. "등록" 버튼을 클릭합니다.
4. ⚠️ 동일한 기기명 또는 기기 코드는 사용할 수 없습니다.

**기기 상태:**
| 상태 | 설명 |
|------|------|
| 정상 (NORMAL) | 생산 배정 가능 |
| 정비 중 (MAINTENANCE) | 일시적 사용 불가 |
| 고장 (BROKEN) | 완전 사용 불가 |

**기기 삭제:**
- 기기 상세 페이지에서 삭제할 수 있습니다.
- 진행 중인 예약이 배정된 기기는 삭제할 수 없습니다.

### 4.5 고객사 관리 (슈퍼 관리자 전용)

**고객 목록:**
- 전체 고객사 목록, 상태(활성/정지/미승인) 확인
- 검색 및 상태별 탭 필터

**고객 등록:**
1. "고객 등록" 버튼을 클릭합니다.
2. 이메일, 비밀번호, 담당자명, 업체명, 연락처를 입력합니다.
3. "고객 등록" 버튼을 클릭합니다.
4. 생성된 이메일/비밀번호를 고객에게 전달합니다.

**고객 상세:**
- 고객 정보, 예약 이력 확인
- 계정 정지 / 정지 해제 / 삭제 기능

### 4.6 직원 관리 (슈퍼 관리자 전용)

**직원 등록:**
1. "직원 등록" 버튼을 클릭합니다.
2. 이메일, 비밀번호, 이름, 역할(관리자/작업자), 연락처를 입력합니다.
3. "직원 등록" 버튼을 클릭합니다.

**직원 관리:**
- 상태별 탭 필터(전체/관리자/작업자)
- 계정 비활성화 / 삭제 기능

### 4.7 문의 관리

고객이 접수한 문의를 확인하고 답변합니다.

1. 문의 목록에서 미답변 건을 확인합니다.
2. 문의를 클릭하여 상세 내용을 확인합니다.
3. 답변을 입력하고 "답변 등록" 버튼을 클릭합니다.
4. 고객에게 답변 완료 알림이 발송됩니다.

## 5. 작업자 매뉴얼

### 5.1 대시보드

오늘의 생산 작업을 한눈에 확인합니다.

**통계 카드:**
- 오늘 총 작업: 오늘 예정된 전체 작업 건수 및 총 중량
- 생산 중: 현재 진행 중인 작업 건수
- 완료: 오늘 완료된 작업 건수
- 내일 예정: 내일 예정된 작업 건수

**날짜 선택:**
- 상단 날짜 선택기로 다른 날짜의 작업을 확인할 수 있습니다.
- "오늘" 버튼으로 오늘 날짜로 돌아올 수 있습니다.

**기기 필터:**
- 전체/소형/중형/대형/기타 탭으로 기기 종류별 작업을 필터링합니다.

**작업 목록:**
| 열 | 설명 |
|-----|------|
| 기기 | 배정된 기기명 |
| 업체 | 고객사명 |
| 제품 | 제품명 |
| 중량 | 원두 중량(kg) |
| 상태 | 확정/생산 중/완료 |
| 작업 | 출고 처리 버튼 (생산 중일 때) |

**이번 주 사이드바:**
- 우측에 이번 주 요일별 작업 건수와 중량이 표시됩니다.
- 휴무일은 별도 뱃지로 표시됩니다.
- 날짜를 클릭하면 해당 날짜로 이동합니다.
- 하단의 "생산 이력 전체 보기"를 클릭하면 생산 이력 페이지로 이동합니다.

### 5.2 생산 이력

월별 생산 현황을 캘린더 형태로 확인합니다.

**월간 통계:**
- 이번 달 생산 건수
- 총 중량 (kg)
- 완료율 (%)

**캘린더:**
- 각 날짜에 생산 건이 기기명과 함께 표시됩니다.
- 중량(kg) 뱃지가 함께 표시됩니다.
- 휴무일은 회색 뱃지로 표시됩니다.
- 날짜를 클릭하면 해당 날짜의 상세 작업 목록을 확인합니다.
- 좌우 화살표로 월을 이동할 수 있습니다.

### 5.3 출고 관리

생산이 완료된 제품의 출고를 처리합니다.

**출고 목록:**
- 상태별 탭: 전체/생산 중/생산 완료/출고 완료
- 검색: 업체명, 제품명, 예약번호로 검색
- 날짜 필터: 시작일~종료일 범위 지정

**출고 처리:**
1. 출고 목록에서 "생산 완료" 상태인 항목의 "출고 처리" 버튼을 클릭합니다.
2. 출고 상세 페이지에서 생산 주문 정보를 확인합니다.
3. 택배사를 선택합니다 (CJ대한통운, 한진택배, 롯데택배, 우체국택배, 로젠택배, 직접 입력).
4. 송장번호를 입력합니다.
5. 우측의 카카오 알림톡 미리보기에서 고객에게 발송될 내용을 확인합니다.
6. "출고 처리 완료" 버튼을 클릭합니다.

**출고 완료 확인:**
- 출고 완료된 항목의 "상세" 버튼을 클릭하면 택배사와 송장번호를 다시 확인할 수 있습니다.

## 6. 부록

### 6.1 자동 상태 전환

시스템은 매일 자동으로 예약 상태를 전환합니다.

| 시간 | 동작 |
|------|------|
| 정오 (12:00) | "확정" → "생산 중" (당일 예약) |
| 12:01 | "생산 중" → "완료" (전일 예약) |

- 고객에게는 생산 시작 전 알림이 발송됩니다 (11:59).

### 6.2 자동 배정 규칙

예약 신청 시 시스템이 자동으로 최적의 기기를 배정합니다.

**배정 우선순위:**
1. 단일 기기로 처리 가능하면 해당 기기에 배정
2. 대형 → 중형 → 소형 순서로 분산 배정
3. 21~24kg 데드존: 중형(20kg) + 소형(나머지) 강제 분할
4. 캔 제품은 소형 기기 배정 불가

### 6.3 팝업 이동

모든 팝업(모달) 창은 상단 타이틀 바를 마우스로 잡고 드래그하여 위치를 이동할 수 있습니다. 팝업을 닫으면 위치가 초기화됩니다.
